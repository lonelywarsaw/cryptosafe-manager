# CryptoSafe Manager

Менеджер паролей с шифрованной базой и графическим интерфейсом (PyQt6).

## Что сделано

- **Первый запуск** — мастер настройки: ввод мастер-пароля и выбор папки для базы (SQLite).
- **Разблокировка** — вход по мастер-паролю в уже созданную базу.
- **Хранилище** — добавление, редактирование и удаление записей (сайт, логин, пароль, заметки). Пароли в базе хранятся в зашифрованном виде (заглушка шифрования, полноценное шифрование — в следующих спринтах).
- **Настройки** — тема (светлая/тёмная/системная), язык (русский/английский), таймаут буфера обмена и авто-блокировка. Настройки сохраняются в файл и при открытой базе — в таблицу настроек.
- **Аудит** — просмотр лога действий (добавление, изменение, удаление записей).
- **Резервная копия** — сохранение копии файла базы на диск.
- **Тесты** — модульные и интеграционные тесты, в CI проверяется покрытие кода (≥70%).

## Как запустить

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Запустить программу из корня проекта:

```bash
python main.py
```

При первом запуске откроется мастер настройки — задайте мастер-пароль и выберите, где хранить базу. Дальше можно добавлять записи через меню «Правка» → «Добавить», открывать настройки и лог через «Вид».

## Запуск тестов

```bash
python -m unittest discover -s tests -p "*test*.py" -v
```
